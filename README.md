# ComicStar - Онлайн-библиотека комиксов

## Тема проекта
Онлайн-библиотека комиксов "ComicStar". Проект представляет собой минималистичный лендинг для сервиса чтения комиксов онлайн.

## Использованный фреймворк
**Pure.css** - - легковесный CSS-фреймворк для стилизации и сеток

## Структура проекта
- `index.html` - Главная страница с лендингом
- `catalog.html` - Каталог комиксов
- `faq.html` - Страница с часто задаваемыми вопросами (аккордеон)
- `contacts.html` - Контактная форма
- `404.html` - Страница ошибки 404
- `marvel/ultimate-spider-man.html` - Страница комикса (Ultimate Spider-Man)
- `dc/` - Раздел DC (с 404 страницей в разработке)
